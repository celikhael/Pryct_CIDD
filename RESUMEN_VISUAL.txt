🔌 API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════

GET /
└─ Retorna el dashboard HTML

GET /api/metrics
└─ Retorna: accuracy, precision, recall, f1, auc, confusion_matrix

└─ Retorna: top 15 características con su importancia

🔧 PERSONALIZACIÓN
═══════════════════════════════════════════════════════════════════════════

Cambiar Colores:
- En la sección :root del CSS
- --color-highlight: #e94560
- --color-success: #00d4ff
- --color-warning: #ffd700

Cambiar Puerto:
- Editar última línea de dashboard.py
- app.run(debug=True, host='0.0.0.0', port=8080)

Cambiar Frecuencia de Actualización:
- En el JavaScript: setInterval(loadData, 30000);
- Cambiar 30000 ms (30 segundos)

Cambiar Fuentes:
- En @import de Google Fonts
- En CSS variables --font-display y --font-mono
